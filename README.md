# Bevy Snippets
[Visual Studio Code](https://code.visualstudio.com/) extension that adds snippets for the [Bevy Engine](https://bevyengine.org/).

## Snippets
* ### Parameters
    - `query`
    - `query mut`
    - `commands`
    - `res`
    - `res mut`
    - `event writer`
    - `event reader`
    - `local`
* ### Templates
    - `app`
    - `plugin`
* ### Structs
    - `component`
    - `resource`
